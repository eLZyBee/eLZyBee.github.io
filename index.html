ChoreBunny

Full stack web application emulating the services of TaskRabbit, with a twist.

ChoreBunny is deployed on Heroku, using a Ruby on Rails back-end and PostgreSQL database. The front-end is written in JavaScript, using React and Flux. jQuery is utilized for aspects of the display and making AJAX queries.

ChoreBunny allows parents to assign chores to their children in exchange for the accumulation of BunnyPoints™.
